## Requirements

* Node (8.11.3)
* Yarn (1.9.4)

## Install

yarn install <br>

## Usage

#### Run local server

Watch the src files: ````yarn run build-dev```` <br>
Start the server:    ````yarn run start````

#### Build production and run server

yarn run build <br>
yarn run start

#### Configuration

Application wide configuration files reside in ````config/default.js````
