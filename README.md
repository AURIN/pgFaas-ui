Front-end interface for FaaS on PostGIS

## Table of Contents

## Requirements

* Node (8.11.3)
* Yarn (1.9.4)

## Install

yarn install <br>

## Usage

#### Run local server

yarn run build-dev <br>
yarn run start

#### Build production

yarn run build <br>
yarn run start

#### Configuration

Application wide configuration files reside in ````config/default.js````
